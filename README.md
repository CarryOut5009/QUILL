Overview
QUILL is a privacy-focused writing assistant built as a capstone project for an AI Concentration program. It provides three core features while keeping all your data local on your computer:
The 3 Pillars

Grammar & Style Correction (Pillar 1)

Real-time grammar and spelling detection
Punctuation error correction
Right-click context menu with suggestions
Detailed explanations for each error


Intelligent Text Generation (Pillar 2)

Continue writing from cursor position
Paraphrase entire documents
Rewrite text for clarity
Context-aware suggestions


Tone & Purpose Adjustment (Pillar 3)

Transform to Formal business language
Convert to Casual friendly style
Technical/scientific terminology mode
Simplify complex text

Privacy First
Unlike cloud-based tools like Grammarly or ChatGPT:

100% Local Processing - All AI runs on your computer
No Internet Required - Core features work offline
No Data Upload - Your documents never leave your device
Local Database - Secure SQLite storage
No Tracking - No analytics or telemetry

Perfect for:

Confidential documents
Sensitive research
Private journaling
Proprietary business writing
Anyone who values digital privacy
